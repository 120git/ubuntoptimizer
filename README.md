A comprehensive system optimization suite for Ubuntu and other Linux distributions.

## Features

- Full system optimization and maintenance
- Interactive menu interface
- System health monitoring
- Performance benchmarking
- Backup functionality
- Advanced package management
- Detailed logging system
- SSD/NVMe optimization
- Memory management optimization
- Service monitoring and optimization

## Requirements

- Root/sudo privileges
- Bash shell
- Common Linux utilities (curl, wget, htop, lsof, iotop)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/120git/ubuntoptimizer.git
cd ubuntoptimizer
```

2. Make the script executable:
```bash
chmod +x system-optimize.sh
```

## Usage

Run the script with sudo:
```bash
sudo ./system-optimize.sh
```

The script will present an interactive menu with the following options:
1. Run Full System Optimization
2. Update System Only
3. Optimize System Only
4. Show System Information
5. Check System Health
6. Create System Backup
7. Run System Benchmark
8. View Logs
9. Exit

## Logs and Backups

- Logs are stored in `/var/log/system-optimize.log`
- System backups are stored in `/var/backups/system-optimize/`

## Supported Distributions

- Ubuntu
- Debian
- Fedora
- RHEL/CentOS
- Arch Linux/Manjaro
